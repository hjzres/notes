Review

1. Combining Like Terms

  • A number, Variable, power of a variable, product of a number + power of a variable = Term
  • The number that is multiplied by the variable in a term = COEFFICIENT
  • A letter that represents a value that can change or vary in an expression = variable
  • A non-varying value = Constant
  • Terms that contain the same variable raised to the same exponent = Like Terms
  • Terms that contain different variables or contain the same variable raised to different exponents = Unlike Terms

When collecting like terms, add or subtract the coefficients and leave the variable and its exponent alone:

a) 5x + 3x - 6
=8x - 6
b) -6 - 3 - 2x² + 5x - 7x²
= -9x² + 5x - 9


2. Distributive property

Multiply the Term that is outisde of a bracket into each of the terms inside the bracket

Careful with Signs

a) 5(3x + 4)
= 15x + 20
b) -6(7x - 9)
= -42x + 54
c) -3(-8-6x)
= 18x + 24

3. Solving Equations

  1. Maneuver all like terms on one side of the equal sign(the variable should always be on the LEFT side!)
  2. collect like terms
  3. Isolate(or solve for) the variable by doing reverse BEDMAS
a) -5 = x - 4
x = -5 + 4
x = -1
b) -49 = 7x
-49/7 = 7x/7
x = 7
c) x/8 = -9
x = 72
